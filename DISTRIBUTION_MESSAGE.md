# 配布時のメッセージテンプレ

LINE / メール / Slack で配布相手に送る文面のサンプル。

---

## 📋 シンプル版（コピペ用）

```
Kindleで買った本を、PCで検索できるPDFとMarkdownに変換するアプリを作りました。

【ダウンロード】
https://github.com/f2t-code/kindle-ocr-app/releases/latest

【Macの方】
1. KindleOCR-Mac.zip をDL → ダブルクリックで解凍
2. アプリを「右クリック → 開く → 開く」（初回のみ）
3. macOSが権限を聞いてきたらすべて「許可」

【Windowsの方】
1. KindleOCR-Windows.zip をDL → 解凍
2. Kindle OCR.exe をダブルクリック
3. SmartScreen警告 → 「詳細情報 → 実行」
4. ウイルス対策ソフトが遮断したら除外設定（詳細はガイド参照）

【詳しい使い方】
同梱の USER_GUIDE.md を見てください。

【注意】
- 個人で読むためだけに使用してください
- 配布・共有・転載は著作権法違反になります

詰まったらスクショと一緒に連絡ください。
```

## 🛡️ Windows ユーザーが多い場合の追加メッセージ

```
※ Windowsの方へ補足

アンチウイルスソフト（Avast/Norton/Defender等）が
「pyautogui」というキー操作ライブラリを「キーロガーかも？」と
誤検知することがあります。

その場合は以下のいずれかで対処してください:

【Windows Defender】
スタート → 「Windows セキュリティ」を開く
→ ウイルスと脅威の防止 → 設定の管理
→ 除外の追加または削除 → 「+除外の追加」→ フォルダー
→ Kindle OCRフォルダを指定

【Avast】
Avast → 設定 → 例外 → ファイル/フォルダパスを追加
→ Kindle OCRフォルダを指定

その他のアンチウイルスでも、似たような「除外設定」があります。

それでも詰まったら連絡ください。
```

## 📞 個別フォロー用テンプレ

### 「権限ポップアップを拒否しちゃった」と言われたら

```
Mac の方ですか?

1. アプリ画面の上の方にある「⚙️ アクセシビリティ設定を開く」を押す
2. システム設定が開くので、リストに「Kindle OCR」がある
3. その右側のトグルスイッチを ON にする
4. アプリを ⌘Q で終了 → もう一度ダブルクリック起動

同じ手順で「画面収録」も ON にしてください。
```

### 「ウイルスかも？怖い」と言われたら

```
ご安心ください。怖く感じる原因は2つあります:

1. 「開発元未確認」警告
   Apple/Microsoftに開発者登録（有料）していないため出る警告です。
   PCが「知らない人が作ったアプリだから一応警告するね」と言ってるだけ。
   無害です。

2. アンチウイルスの誤検知
   このアプリは「キーボード操作を自動化する」ライブラリを使うため、
   一部のアンチウイルスは「キーロガーかも？」と誤検知します。
   実際にはキーロガーではなく、Kindleのページ送り→キーを送るだけ。

ソースコードは公開しているので、心配なら見てみてください:
https://github.com/f2t-code/kindle-ocr-app

それでも不安なら、私が画面共有でセットアップ一緒にやります。
```

### 「ページがめくれない」と言われたら

```
よくある原因と対処:

① Mac の場合
「アクセシビリティ」権限が許可されているか確認:
システム設定 → プライバシーとセキュリティ → アクセシビリティ
→ Kindle OCR のトグルが ON か?

② Windows の場合
アンチウイルスが pyautogui を遮断している可能性:
Kindle OCRフォルダをアンチウイルスの除外設定に追加

③ 両OS共通
Kindleアプリが最前面にちゃんと切り替わっているか?
他のウィンドウが上に被っていないか?

それでもダメなら、診断テストの結果を送ってください。
```
